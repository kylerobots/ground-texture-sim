Position the camera at a designated pose and render an image.

This method takes the 6 DOF pose of the camera, places the camera at that pose in the environment, renders the image, then saves the image to the designated file.

Blender's coordinate system is different than the usual coordinate system assumed in robotics. This method adds the correct adjustment, so there is no need to account for it. In other words, specify the camera pose using conventional robotics coordinate systems.

Parameters
image_pathAn absolute path to where the image should go. Blender does not seem to handle relative paths great.
camera_poseThe 4x4 homogenous matrix representing the pose of the camera in the world frame.
Returns
None
Exceptions
ValueErrorraised if the provided path is not absolute.

97 other words, specify the camera pose using conventional robotics coordinate systems.
98
99 @param image_path An absolute path to where the image should go. Blender does not seem to
100 handle relative paths great.
101 @param camera_pose The 4x4 homogenous matrix representing the pose of the camera in the
102 world frame.
103 @return None
104 @exception ValueError raised if the provided path is not absolute.
105 """
106 # Verify the path is absolute.
107 if not path.isabs(image_path):
108 raise ValueError(
109 F'Image path must be absolute. Received: {image_path}')
110 # Convert the provided pose into a Mathutils matrix. This has better support for extracting
111 # the XYZ, RPY values needed to position in Blender.
112 pose = mathutils.Matrix()
113 for i in range(4):
114 for j in range(4):
115 pose[i][j] = camera_pose[i][j]
116 # Calculate the offset to get Blender to position the camera correctly.
117 # Pre-compute the offset used to get blender camera poses correct.
118 blender_adjustment = mathutils.Euler(
119 (pi/2.0, 0.0, -pi/2.0), 'XYZ').to_matrix().to_4x4()
120 blender_placement = pose @ blender_adjustment
121 # Now place the camera
122 camera = bpy.data.objects[self.camera_name]
123 camera.location.x = blender_placement.to_translation()[0]
124 camera.location.y = blender_placement.to_translation()[1]
125 camera.location.z = blender_placement.to_translation()[2]
126 camera.rotation_euler.x = blender_placement.to_euler('XYZ')[0]
127 camera.rotation_euler.y = blender_placement.to_euler('XYZ')[1]
128 camera.rotation_euler.z = blender_placement.to_euler('XYZ')[2]
129 # Render the image
130 bpy.context.scene.render.filepath = image_path
